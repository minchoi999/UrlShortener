Whenver you are adding image files on websites, make sure you either use jpg or png format,
and when you are using an image as a logo or background-image, make sure to apply
alpha filter on the image to make the background transparent.

How to apply alpha filter on an image using GIMP (Open source Photoshop)
1. Open the image file you want to edit
2. Click Colors in the main context menu
3. Choose Color to Alpha
3.1 preview the image, and click OK
4. Click File in the main context menu
4.1 Click Export As and choose png as the file type
